# _Advanced Weaponry Interaction_

#### • Upgrades armory handling with a system that moves beyond simple menus. This script replaces standard "press button to receive" prompts with high-quality, authentic animations and immersive feedback for a superior tactile experience.

#### • Physical interaction: No more static UI prompts—interact with weapons through realistic character animations.

#### • Immersive feedback: Designed to provide a tangible sense of weight and action when handling weaponry.

#### • Database driven: Every interaction and item state is securely tracked and saved in the database to avoid unlimited weapons.

# _Fortgeschrittenes Lagersystem_

#### • Erweitert das Lagersystem durch ein System, das weit über einfache Menüs hinausgeht. Dieses Script ersetzt standardmäßige "Knopf drücken" prompts durch hochwertige, authentische Animationen und immersives Feedback für ein erstklassiges Spielgefühl.

#### • Physische Interaktion: Keine statischen UI-Meldungen mehr. Die Interaktion mit Waffen erfolgt über realistische Charakter-Animationen.

#### • Immersives Feedback: Entwickelt, um beim Umgang mit Waffen ein spürbares Gefühl für Gewicht und Aktion zu vermitteln.

#### • Datenbankgestützt: Jede Interaktion und jedes Item wird sicher in der Datenbank erfasst und gespeichert, damit keine unendliche Waffen rumgehen.
