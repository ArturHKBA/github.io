# _Experimental Deformation Simulation_

#### • This project was an exploration into high-fidelity vehicle physics, focusing on real-time deformation. Developed for older versions of Unity, it aimed to bridge the gap between arcade driving and realistic simulation through visual and physical feedback.

#### • Dynamic deformation: Simulates the compression and expansion of wheels based on vehicle weight and surface impact.

#### • Physics-Driven: Designed to react to different forces and suspension loads for a better driving feel.

#### • Experimental tech: A "passion project" created to test the limits of Unity’s physics engine regarding soft-body approximations.

### Status Notice:

#### • Legacy code: This script is currently deprecated and optimized for older Unity versions.

#### • Unfinished: While functional as a standalone simulation, it was never integrated into a final production environment.

# _Experimentelle Deformations-Simulation_

#### • Dieses Projekt war eine technische Studie im Bereich der Fahrzeugphysik mit dem Fokus auf Echtzeit-Reifendeformation. Entwickelt für ältere Unity-Versionen, war das Ziel, die Lücke zwischen Arcade-Fahrverhalten und realistischer Simulation durch visuelles und physisches Feedback zu schließen.

#### • Dynamische Verformung: Simuliert das Stauchen und Dehnen der Reifenflanken basierend auf Fahrzeuggewicht und Untergrund.

#### • Physikbasiert: Reagiert auf Federungslasten, um ein greifbareres Fahrgefühl zu vermitteln.

#### • Experimentelle Technik: Ein Project, um die Grenzen der Unity Physics Engine im Bereich von Soft Body zu testen.

### Status-Hinweis:

#### • Legacy-Code: Dieses Script ist aktuell veraltet (deprecated) und auf ältere Unity-Versionen optimiert.

#### • Unvollendet: Obwohl es als Simulation funktionsfähig ist, wurde es nie in eine finale Produktionsumgebung integriert. Es dient jedoch als starke Basis für zukünftige Physik und Entwicklung.
