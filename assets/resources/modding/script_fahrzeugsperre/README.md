# _Vehicle Locking System_

### • This script provides a secure and database-driven vehicle locking system for vehicles. It ensures that only authorized owners can interact with their vehicles based on unique identifiers.

### • Toggle locks: Ability to lock and unlock vehicles.

### • Ownership verification: Integrates with your database to ensure the vehicle is registered to the player.

### • Inventory integration: Requires a physical key item in the player's inventory to function.

### • License plate matching: Automatically links vehicle license plates to unique Key IDs upon purchase for seamless authentication.

### • When a vehicle is purchased, the script generates a record in the database using the license plate. To unlock the car, the script performs a server-side check to verify that the Key ID in the player's inventory matches the License Plate stored in the database.

# _Fahrzeug Schließsystem_

### • Dieses Script bietet ein sicheres, Datenbank gestütztes Schließsystem für Fahrzeuge. Es stellt sicher, dass nur autorisierte Besitzer basierend auf eindeutigen Identifikatoren mit ihren Fahrzeugen interagieren können.

### • Schließfunktion: Möglichkeit, Fahrzeuge zu sperren und zu entsperren.

### • Besitzprüfung: Integration mit der Datenbank, um den rechtmäßigen Besitz durch den Spieler zu verifizieren.

### • Inventar-basiertes System: Die Funktion setzt voraus, dass sich ein physischer Fahrzeugschlüssel im Inventar des Spielers befindet.

### • Kennzeichen-Abgleich: Verknüpft beim Fahrzeugkauf automatisch das Kennzeichen mit einer eindeutigen Schlüssel-ID für eine Authentifizierung.

### • Sobald ein Fahrzeug gekauft wird, speichert das Script einen Datensatz mit dem Kennzeichen in der Datenbank. Beim Schließvorgang prüft das Script serverseitig, ob die Schlüssel-ID aus dem Inventar des Spielers mit dem Kennzeichen in der Datenbank übereinstimmt.
