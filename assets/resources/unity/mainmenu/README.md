# _Dynamic Settings & Graphics Menu_

#### • A lightweight and efficient settings framework that allows for real-time engine adjustments. It provides a clean interface for players to optimize their visual experience mostly without restarting the game.

#### • Preset switching: Quickly toggle between pre-defined configurations (Potato, Very low, Low, Medium, High, Ultra, Cinematic, Frame Breaking etc.).

#### • Runtime modification: Directly modifies engine properties and graphical parameters on the fly.

#### • User-Friendly interface: A simple, high-performance menu designed for seamless integration.

#### • Performance focused: Optimized to ensure that applying settings doesn't cause significant frame drops.

# _Dynamisches Grafik & Einstellungsmenü_

#### • Ein leichtgewichtiges und effizientes Framework für Systemeinstellungen, das Anpassungen der Engine in Echtzeit ermöglicht. Es bietet Spielern eine klare Oberfläche, um die visuelle Performance meist ohne Neustart zu optimieren.

#### • Preset-Steuerung: Schnelles Umschalten zwischen vordefinierten Konfigurationen (Kartoffel, Sehr Niedrig, Niedrig, Mittel, Hoch, Ultra, Cinematic, Frame Breaking etc.).

#### • Echtzeit-Anpassung: Direkte Modifikation von Engine Eigenschaften und Grafikparametern zur Laufzeit.

#### • Benutzerfreundliche UI: Ein simples, performantes Menü für eine nahtlose Integration in bestehende Projekte.

#### • Performance-Optimiert: Sicherstellung, dass das Übernehmen von Einstellungen keine signifikanten Framedrops verursacht.
