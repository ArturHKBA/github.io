# _Realistic randomized fire/flame spread script_

#### • This script allows whitelisted users to generate and manage realistic fire scenarios designed for realistic/hardcore roleplay. it features advanced extinguishing mechanics and deep integration with my emergency service jobs.

#### • Whitelist access: Only whitelisted players have the permission to trigger and create fire events/scenarios.

#### • Realistic interactions: Fires and flames are fully interactable and can be extinguished using fire extinguishers or custom hose/water systems (via Hash).

#### • Scripted scenarios: Includes a randomizer that spawns fires at predefined, saved positions to create dynamic world events/scenarios.

#### • Job integration: Fully compatible and synced with the existing Firefighter Job Script.

# _Realistisches zufallsgeneriertes Feuer/Flammenausbreitung Script_

#### • Dieses script ermöglicht es autorisierten Spielern, realistische Brandszenarien zu erstellen und zu verwalten. Es wurde speziell für anspruchsvolles Roleplay entwickelt und bietet fortschrittliche Löschmechaniken sowie eine tiefe Integration in Rettungsdienst-Jobs.

#### • Whitelist Zugriff: Nur freigeschaltete Spieler haben die Berechtigung, Brandereignisse per commands auszulösen.

#### • Realistische Löschmechanik: Feuer und Flammen reagieren physikalisch korrekt auf Feuerlöscher sowie auf benutzerdefinierte Schläuche/Wassersysteme (via Hash).

#### • Zufalls-Szenarien: Ein Randomizer sorgt dafür, dass Brände an zuvor gespeicherten Koordinaten entstehen, um dynamische Einsätze zu generieren.

#### • Job Kompatibilität: Vollständig kompatibel und synchronisiert mit dem vorhandenen Feuerwehr Job Script.
