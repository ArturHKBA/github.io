# _VR Optimization & Localization Tools_

#### • A utility-focused VR toolkit designed to improve performance and accessibility. It provides players with direct control over the world environment while ensuring a global reach through multilingual support.

#### • Performance panel: Interactive in-game toggle to disable heavy objects for an instant FPS boost.

#### • Prefab reset: Easily reset specific objects or prefabs back to their original positions.

#### • Global accessibility: Fully localized interface and world elements.

#### • Multilingual support: Designed to support a wide range of languages, ensuring everyone can navigate the world with ease.

# _VR-Optimierungs & Lokalisierungstools_

#### • Ein nutzerorientiertes VR-Toolkit zur Verbesserung der Performance und Barrierefreiheit. Es gibt Spielern direkte Kontrolle über die Spielwelt und ermöglicht durch Mehrsprachigkeit eine globale Reichweite.

#### • Performance-Panel: Interaktive Schaltflächen zum Deaktivieren von Objekten für einen sofortigen FPS-Boost.

#### • Prefab-Reset: Setzt spezifische Objekte oder Prefabs einfach auf ihre ursprüngliche Position zurück.

#### • Globale Barrierefreiheit: Vollständig lokalisierte Benutzeroberfläche und Welt-Elemente.

#### • Mehrsprachige Unterstützung: Entwickelt für eine Vielzahl von Sprachen, damit sich jeder in der Welt problemlos zurechtfinden kann.
