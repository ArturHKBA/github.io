# _Masken-Shop System_

#### • Steigert die Immersion auf deinem Server mit einem spezialisierten Shop für den Kauf und die Verwaltung von Masken. Jede Maske wird als eigenständiges Item in der Datenbank gespeichert, was dauerhaften Besitz und eine saubere Synchronisation garantiert.

#### • Custom blip (MapMarker): Der Shop ist durch einen benutzerdefinierten Blip leicht auf der Karte zu finden.

#### • Item-basiertes System: Jede Maske wird als Item in der Datenbank registriert.

#### • Vollständig Synchronisiert: Echtzeit-Abgleich, damit alle Spieler die Masken korrekt sehen.

### Für die Zukunft: Die code-Struktur ermöglicht umfassende Anpassungsmöglichkeiten in kommenden Updates.
