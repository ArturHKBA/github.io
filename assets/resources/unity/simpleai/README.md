# _Simple NavMesh AI System_

#### • A core AI framework built using Unity’s NavMesh system. It provides a reliable foundation for NPC movement, patrol logic and basic combat engagement.

#### • Patrol logic: The AI automatically cycles through pre-defined waypoints (A > B > C > A) to secure an area.

#### • Navigation: Uses Unity NavMesh to intelligently understand walkable surfaces and avoid obstacles.

#### • Field of View (FoV): If the player enters the AI's line of sight, it immediately breaks its patrol.

#### • Chase & Attack: The AI chases the player until it reaches a specific distance, at which point it triggers an attack.

# _Simples NavMesh KI-System_

#### • Ein grundlegendes KI-Framework, das auf dem NavMesh-System von Unity basiert. Es bietet eine solide Basis für NPC-Bewegungen, Patrouillen-Logik und einfache Kampf-Interaktionen.

#### • Patrouillen-Logik: Die KI läuft automatisch vordefinierte Wegpunkte ab (A > B > C > A), um ein Gebiet zu bewachen.

#### • Navigation: Nutzt Unity NavMesh, um begehbare Flächen zu erkennen und Hindernisse zu umgehen.

#### • Sichtfeld (FoV): Sobald der Spieler in das Sichtfeld der KI tritt, wird die Patrouille unterbrochen.

#### • Verfolgung & Angriff: Die KI verfolgt den Spieler, bis eine festgelegte Distanz erreicht ist, um dann einen Angriff auszulösen.
