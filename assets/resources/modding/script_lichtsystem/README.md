# _Advanced Emergency Lighting & Siren System_

#### • This script revolutionizes emergency vehicle operations by moving beyond the limited "single-button" vanilla mechanics. It introduces a multi-state control system for a truly professional roleplay experience.

#### • Independent light Control: Use emergency lights without sirens for silent responses or scene security.

#### • Multi-State siren system: Replaces the basic "on/off" toggle with a layered system for manual control over specific tones.

#### • Customizable patterns: Access different flash patterns and speeds to match the urgency of the situation.

### Technical Implementation:

#### • We have completely rebuilt the interaction logic from the ground up. Instead of simple toggles, this script tracks multiple states, allowing players to cycle through siren modes and lighting phases independently.

# _Blaulicht & Sirenen-System_

#### • Dieses Script revolutioniert den Einsatz von Einsatzfahrzeugen, indem es die limitierten "Ein-Tasten-Mechaniken" aus Vanilla durch ein komplexes Multi-State-System ersetzt. Es bietet eine professionelle Steuerung für realistisches Roleplay.

#### • Unabhängige Lichtsteuerung: Nutze das Blaulicht ohne Sirene für stille Anfahrten oder zur Absicherung von Einsatzstellen.

#### • Mehrstufiges Sirenensystem: Ersetzt das simple "An/Aus" Toggle Prinzip durch ein Schicht System zur manuellen Auswahl spezifischer Töne.

#### • Anpassbares Lichtm: Zugriff auf verschiedene Blitze und Geschwindigkeiten, passend zur jeweiligen Einsatzlage.

#### Technische Umsetzung:

#### • Die Interaktionslogik wurde von Grund auf neu entwickelt. Anstatt einfaches Umschalten trackt dieses Script mehrere Zustände gleichzeitig, was es Spielern ermöglicht, Sirenenmodi und Lichtphasen unabhängig voneinander zu steuern.
